Local Video Wall Playlist

LocalVideoWallPlaylist is a single html static file that allows you to specify a few local video files and it display a wall of videos. Nothing is uploaded to any server. Everything is kept locally in your browser. This works offline if you save the page first.

This is made with a different twist from the other video wall of F² Microtools suite as it uses a fixed grid and manages a repeating playlist rather than fitting all of the videos on the wall at once

There's not many bells or whisles. This was just a little experiment to build the most lightweight tool for this without using external dependencies.

Pull requests to improve or add functionalities are welcome.
